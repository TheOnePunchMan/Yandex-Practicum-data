# Репозиторий выпускника профессии "Специалист по Data Science"


## Описание проектов

Данные проекты были выполнены в ходе обучения в Яндекс.Практикуме, по профессии "Специалист по Data Science".

| Название проекта | Описание | Используемые библиотеки | 
| :---------------------- | :---------------------- | :---------------------- |
| [Музыка больших городов](big_cities_music) | Сравнение предпочтений пользователей Яндекс.Музыки из Москвы и Санкт-Петербурга в зависимости от времени (утро и вечер) и дня недели (понедельник, среда, пятница)| *pandas* |
| [Исследование надёжности заёмщиков](bank_preprocessing) | Оценка влияния семейного положения и количества детей клиента на факт погашения кредита в срок. | *pandas* |
| [Исследование объявлений о продаже квартир](spb_real_estate) | Определение рыночной стоимости объектов недвижимости в Санкт-Петербурге и соседних населённых пунктах за несколько лет. | *pandas, matplotlib, numpy* |
| [Определение перспективного тарифа для телеком-компании](megaline_traffic) | Проведение предварительного анализа тарифов на небольшой выборке клиентов "Мегалайна" за 2018 год. | *pandas, matplotlib, numpy, scipy* |
| [Выявление определяющих успешность игры закономерностей](console_games) | Прогнозирование продаж консольных игр и популярности консолей на 2017 год. | *pandas, matplotlib, numpy, scipy* |
| [Анализ поведения клиентов и предложение нового тарифа](megaline_tariff_ml) | Обучение моделей машинного обучения на небольшой выборке клиентов "Мегалайна" за 2018 год. | *pandas, matplotlib, seaborn, sklearn* |
| [Прогнозирование оттока клиентов](bank_churn) | Прогнозирование оттока клиентов из банка в ближайшее время на исторических данных о поведении клиентов и расторжении договоров с "Бета-банком". | *pandas, matplotlib, pandas_profiling, sweetviz, sklearn* |
| [Определение наиболее прибыльного региона для добычи нефти](oil_well_prediction) | Определение региона с максимальной суммарной прибылью нефтяных скважин. | *pandas, matplotlib, sweetviz, sklearn* |
| [Предсказание коэффициента восстановления золота из золотосодержащей руды](gold_content_prediction) | Предсказание коэффициента восстановления золота из золотосодержащей руды. В распоряжении данные с параметрами добычи и очистки. Модель поможет оптимизировать производство, чтобы не запускать предприятие с убыточными характеристиками.| *pandas, matplotlib,  sklearn* |
| [Алгоритм защиты персональных данных пользователей](client_data_encoder) | Нужно защитить данные клиентов страховой компании "Хоть потоп". Разработаем такой метод преобразования данных, чтобы по ним было сложно восстановить персональную информацию. Обоснуем корректность его работы.| *pandas, numpy, sklearn* |
| [Предсказание рыночной стоимости автомобиля](car_prices_prediction) | Построим модель, которая умеет её определять. В вашем распоряжении данные о технических характеристиках, комплектации и ценах других автомобилей.| *pandas, numpy, matplotlib, sklearn* |
| [Прогнозирование количества заказов такси на следующий час](taxi_order_prediction) | Компания «Чётенькое такси» собрала исторические данные о заказах такси в аэропортах. Построим модель, которая умеет определять количество заказов такси в следующий час. В вашем распоряжении данные о технических характеристиках, комплектации и ценах других автомобилей.| *pandas, catboost, matplotlib, sklearn* |
